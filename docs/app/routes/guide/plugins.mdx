## Formatting

This config does not handle code formatting. Following [ESLint's own recommendation](https://eslint.org/blog/2023/10/deprecating-formatting-rules/), formatting is a separate concern best handled by a dedicated tool â€” [Prettier](https://prettier.dev), [Biome](https://biomejs.dev), or [oxfmt](https://oxc.rs).

All formatting-related ESLint rules are disabled at build time so they never conflict with your formatter of choice.
